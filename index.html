Установка

Обзор программ

Fast HTML

Программа для быстрого создания структуры папок для сайта на HTML, удобно использовать в качестве инструмента для ускорения своей работы или обучения.


В index.html сразу подключается css стили и записывается базовая структура html файла. Создается в корне папка assets где находятся папки с файлами для сайта.

Fast PHP

Программа для быстрого создания структуры папок для работы на PHP с автоматической настройкой папки проекта для Open Server, удобно использовать в качестве инструмента для ускорения своей работы или обучения.


В index.php сразу записывается базовая структура html файла где открывается php скрипт. Создается в корне папка assets где находятся папки с файлами для сайта.

Преимущества

Обучение

Можно быстро работать и обучаться. Узнать правильную структуру папок и использовать её.

Простота

Просто запустил програму и все! Создана папка.

Скорость

Вместо того что бы тратить время на создание кучи папок можно нажать два раза на программу и создать новый проект.
